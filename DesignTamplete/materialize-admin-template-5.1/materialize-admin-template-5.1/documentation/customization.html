Template Customization

Materialize Admin provides different types of id, classes & attributes to customize the template.

Body Options

<html class="loading" lang="en" data-textdirection="ltr">
<head></head>
<!--Body -->
<body class="vertical-layout vertical-menu-collapsible page-header-dark vertical-modern-menu 2-columns   menu-expanded fixed-navbar" data-open="click" data-menu="vertical-modern-menu" data-color="" data-col="2-columns">
...
</body>  
<!--/ Body -->
</html>

You can use below mentioned attributes and classes for <body> to customize your application


Attribute Value/Classes Description
class .vertical-layout /
.horizontal-layout 		If you want your application's layout to be vertical you'd have to add .vertical-layout as a class to your <body> tag. If you want your application's layout to be horizontal add .horizontal-layout as a class to your <body> tag. (It is mandatory to add any one of these classes to your body tag)
class .vertical-menu-collapsible/
.horizontal-menu 		There are two type of navigation menu collapsible and horizontal. To get collapisble menu you would have to have add .vertical-menu-collapisble class to your <body> tag. If you want horizontal menu you'd have to add .horizontal-menu (It is mandatory to add any one of these classes to your body tag)
class .page-header-dark /
.page-header-light 		If you want a dark header for your application add .page-header-dark as a class to <body> or if you want a light colored header add class .page-header-light
data-menu vertical-modern-menu /
vertical-menu-nav-dark/
vertical-gradient-menu-template /
vertical-dark-menu-template/
horizontal-menu-template		 data-menu atttribute is just for reference to your <body>
data-open click/ hover If you want your data to be open on click you'd have to add
data-open="click"
attribute or if you want your data to open on hover you'd have to add
data-open="hover"
(It is mandatory to add any one of the following)
Navbar Options
    
<!--Navbar -->
<div class="navbar navbar-fixed"> 
<nav class="navbar-color nav-collapsible sideNav-lock navbar-dark gradient-45deg-indigo-purple no-shadow">
...
</nav>
</div>
<!-- / Navbar -->

You can use below mentioned attributes and classes for <nav> to customize your application


Attribute Value Description
class navbar-fixed/
navbar-static 		To set navbar's postion as fixed use .navbar-fixed as a wrapper of your <nav> tag or if you want your navbar to be static use .navbar-static (It is mandatory to use any one of these classes)
class .navbar-dark /
.navbar-light 		Use class .navbar-dark for a dark colored navbar or if you want a light colored navbar use .navbar-light
class gradient-45deg-purple-deep-orange(any color) You can change color of your navbar by just adding name of color as a class you can pick colors from Colors.
class .gradient-shadow Use gradient shadow to give your navbar a gradient colored shadow
Navigation Menu Options
        
<!--Navigation Menu-->
<aside class="sidenav-main nav-expanded nav-lock nav-collapsible sidenav-light sidenav-active-square">
...
</aside>
<!--/ Navigation Menu-->

You can use below mentioned attributes and classes for <aside> to customize your application


(any color)
Attribute Value/Classes Description
class .sidenav-light /
.sidenav-dark 		You can change you sidenav's color using these classes. If you want a dark sidenav use .sidenav-dark or for light colored sidenav use .sidenav-light (It is mandatory to use one of the following classes)
class .sidenav-active-square /
.sidenav-active-rounded 		Use can select from how you want your active item to look in sidenav by applying these classes to your <aside> tag. if you want rounded active item use .sidenav-active-rounded or for square active item use .sidenav-active-square
class .nav-expanded /
.nav-collapsed 		You can choose if you want your menu collapsed or expanded using .nav-expanded and .nav-collapsed with .nav-lock in your <aside> tag. (It is mandatory to use any one of these classes and you must add class .nav-lock with them)
class .navbar-full If you want the color header of the menu to be same as navbar use .navbar-full class with your <aside> tag.
class .gradient-45deg-purple-deep-orangeYou can change color of the header of menu by using any color as a class you can pick any color from Colors.
Footer Options
            
<!--Footer-->
<footer class="page-footer footer footer-static footer-dark gradient-45deg-indigo-purple gradient-shadow navbar-border navbar-shadow">
...
</footer>
<!--/ Footer-->

You can use below mentioned attributes and classes for <footer> to customize your application


Attribute Value/Classes Description
class .footer-fixed /
.footer-static 		If you want your footer to be fixed you can use class .footer-fixed or if you want footer to be static you can use class .footer-static. (It is mandatory to use any one of the above mentioned class with <footer> tag)
class .footer-dark /
.footer-light 		You can select from dark or light colored footer by these classes with <footer> tag. (It is mandatory to use any one of the following class for footer)
class .gradient-45deg-purple-deep-orange (any color) You can change color of footer by using classes you just have to add name of the color as a class Colors.
